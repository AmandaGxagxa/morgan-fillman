# Tips

## clearfix

Stops the parent element from collapsing.

https://css-tricks.com/snippets/css/clear-fix/

```css

.group:after {
  content: "";
  display: table;
  clear: both;
}

```

## inline-lock

Layout using vertical block;

http://learnlayout.com/inline-block.html

```css
  display: inline-block;
  vertical-align: top;
```
